## 记录各种工具常用的配置

之前我都是把常用的配置放在自己的oneDrive上。偶尔有一天，同事问我为什么我的IDEA配色看起来特别舒服（我其实只是给原生的durala主题加了几个颜色），让我把配置发给他。还有一个原因是我们开发团队都会写一些前端的东西，主要用VSCode。为了代码格式的统一，我们团队还弄出了一个通用的配置文件。经过这两个事件之后我就想把配置文件放在Github上，方便跟其他开发者一起交流。
